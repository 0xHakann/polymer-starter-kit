## Polymer Starter Kit

> A starting point for building web applications with Polymer

### Instructions

With Node and npm installed, run:

```sh
$ npm install -g gulp && npm install -g bower && npm install && bower install
```

### Serve / watch

```sh
$ gulp serve
```

### Run tests

```sh
$ gulp test:local
```

### Build / vulcanize

```sh
$ gulp
```
